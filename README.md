



# Shopping Cart JavaScript - Dress Edition

Welcome to the Shopping Cart project! This application demonstrates a dynamic shopping cart built with JavaScript, allowing users to browse, add, remove, and manage dresses seamlessly. Perfect for e-commerce websites, this project highlights essential JavaScript features like DOM manipulation, event handling, and local storage.

## Features

- Browse a collection of dresses
- Add dresses to the cart
- Remove individual dresses from the cart
- Update dress quantities
- Persist cart data using local storage
- Clear entire cart easily
- Responsive and user-friendly interface

## How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/shopping-cart-dresses.git
   ```
2. Open `index.html` in your browser.
3. Browse dresses and add your favorites to the cart.
4. View and manage your cart in real-time.
